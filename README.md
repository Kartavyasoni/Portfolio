# Kartavya Soni Portfolio

This portfolio showcases Kartavya Soni's work in a luxurious black and white style.
Open `index.html` in a browser to view the site.

## Assets
- `images/hero-avatar.png` – hero image placeholder
- `Kartavya_Soni_Resume.pdf` – downloadable resume

The design loads Google Fonts (Inter and Poppins) and the AOS animation library from their CDNs, so an internet connection is required.
