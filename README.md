# Kartavya Soni Portfolio

This project contains a modern, monochrome portfolio website for Kartavya Soni. Open `index.html` in any browser to view.

## Structure
The site includes sections for Home, About, Skills, Projects, Experience, Leadership, Certificates, and Contact. A sticky navigation bar links to each section.

## Assets
- `Kartavya_Soni_Resume.pdf` – downloadable resume placeholder
- Optional profile photo named `kartavya-portrait.jpg` should be placed in an `images/` folder and will replace the gray placeholder in the hero section.

Google Fonts (Inter, Poppins, and Playfair Display) are loaded from their CDN, so an internet connection is required when viewing the page.

This hero section replicates a split-screen layout inspired by the Morozov-style portfolio UI.
- No external images are used.
- Placeholder box is used instead of profile photo.
- Fonts used: Poppins, Playfair Display, Inter. If not available, fallback is used.
- Responsive for mobile under 768px.
If image or fonts need to be added later, insert filenames and links here.
